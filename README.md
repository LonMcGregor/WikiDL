# Wiki-dl

Script to take a wiki page and turn it into an epub file that can be easily read on an ebook reader.

Requirements:
* Python 3
* EbookLib
* Wikipedia-API

Usage:

```
python wiki_dl.py https://en.wikipedia.org/wiki/Globe globe.epub
```
